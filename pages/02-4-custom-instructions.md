# 02-4. 나만의 ChatGPT 설정하기: 맞춤 설정

## 1. 이 강의에서 배울 내용

이번 강의에서는 ChatGPT 를 나에게 맞게 조정하는 **맞춤 설정** 을 살펴봅니다.

ChatGPT 는 기본 상태로도 사용할 수 있지만, 내가 어떤 사람인지, 어떤 말투를 선호하는지, 어떤 방식의 답변을 원하는지 알려주면 훨씬 더 나에게 맞는 답변을 제공합니다.

이 강의를 통해 다음 내용을 익힐 수 있습니다.

* ChatGPT 맞춤 설정의 목적을 이해할 수 있습니다.
* 개인 맞춤 설정 메뉴에 접근할 수 있습니다.
* 기본 스타일 및 말투를 상황에 맞게 선택할 수 있습니다.
* 따뜻함, 열정적, 헤더 및 목록 사용, 이모지 사용 같은 특성을 조정할 수 있습니다.
* 맞춤형 지침을 사용해 ChatGPT 의 기본 응답 방식을 설정할 수 있습니다.
* 닉네임, 직업, 추가 정보를 입력해 더 개인화된 답변을 받을 수 있습니다.
* 맞춤 설정을 사용할 때 주의해야 할 개인정보와 민감정보 기준을 이해할 수 있습니다.

## 2. 맞춤 설정의 목적

병원 의사는 환자의 상태와 진료 기록을 바탕으로 더 적절한 진료를 합니다.

연인 관계에서도 마찬가지입니다. 상대방의 성향, 취향, 말투, 중요하게 생각하는 것을 잘 알수록 더 자연스럽고 깊이 있는 관계가 만들어집니다.

AI 도 비슷합니다.

ChatGPT 가 사용자에 대해 더 잘 알수록, 더 개인화된 답변과 도움을 제공할 수 있습니다.

예를 들어 사용자가 데이터 분석가라면 ChatGPT 는 분석 업무에 맞는 예시를 더 잘 들 수 있습니다. 강사라면 교육 콘텐츠나 수강생 설명 방식에 맞춰 답변할 수 있습니다. 기업 실무자라면 보고서, 이메일, 회의록, 기획서처럼 실제 업무 문서 중심으로 답변을 구성할 수 있습니다.

맞춤 설정의 목적은 단순합니다.

> ChatGPT 를 나에게 맞는 AI 비서로 조정하는 것입니다.

이 강의에서 다루는 기본 스타일 및 말투, 특성, 맞춤형 지침, 사용자 정보 입력은 모두 **개인화** 를 위한 기능입니다.

## 3. 맞춤 설정 접근 방법

맞춤 설정은 ChatGPT 의 개인화 설정 화면에서 접근할 수 있습니다.

기본 접근 방법은 다음과 같습니다.

1. ChatGPT 에 접속합니다.
2. 화면 좌측 하단의 계정 아이콘을 클릭합니다.
3. `개인 맞춤 설정` 메뉴를 선택합니다.
4. 또는 `설정` 메뉴로 들어간 뒤 `개인 맞춤 설정` 항목을 선택합니다.

브라우저 주소창에서 아래 주소로 바로 접근할 수도 있습니다.

```text
https://chatgpt.com/#settings/Personalization
```

개인 맞춤 설정 화면에서는 다음과 같은 항목을 조정할 수 있습니다.

* 기본 스타일 및 말투
* 특성
* 맞춤형 지침
* 닉네임
* 직업
* 추가 정보
* 메모리 관련 설정

맞춤 설정의 변경 사항은 저장 후 ChatGPT 의 응답 방식에 바로 반영됩니다. 따라서 설정을 바꿨다면 새 채팅에서 간단한 질문을 입력해 실제 응답이 어떻게 달라지는지 확인해 보는 것이 좋습니다.

## 4. 기본 스타일 및 말투

### 4.1 기본 스타일 및 말투란 무엇인가

기본 스타일 및 말투는 ChatGPT 가 어떤 분위기와 성격으로 답변할지를 정하는 설정입니다.

쉽게 말하면 ChatGPT 의 기본 응답 성향을 선택하는 기능입니다.

예를 들어 회사 문서를 작성할 때는 정제되고 격식 있는 답변이 필요할 수 있습니다. 반대로 아이디어를 자유롭게 떠올리는 상황에서는 조금 더 친근하거나 창의적인 톤이 도움이 될 수 있습니다.

즉, 기본 스타일 및 말투는 사용자의 성격을 정하는 것이 아닙니다.

**ChatGPT 가 어떤 스타일로 응답했으면 좋겠는지 지정하는 설정** 입니다.

### 4.2 스타일별 특징

스타일은 사용 환경과 목적에 따라 선택하면 됩니다.

| 스타일       | 응답 방식                           | 기대할 점                      | 추천 상황                  |
| --------- | ------------------------------- | -------------------------- | ---------------------- |
| 프로페셔널     | 격식 있고 정제된 문장, 구조화된 설명, 전문 용어 활용 | 정확한 표현, 비즈니스 언어, 직장에 맞는 격식 | 직장 문서, 공식 커뮤니케이션, 보고서  |
| 친근함       | 따뜻하고 대화체에 가까운 응답                | 공감 표현, 부드러운 설명, 자연스러운 조언   | 아이디어 정리, 자기 성찰, 가벼운 상담 |
| 솔직함       | 직설적이고 핵심 위주의 응답                 | 리스크와 부족한 점을 명확하게 지적        | 계획 점검, 초안 검토, 문제 해결    |
| 엉뚱함       | 장난스럽고 상상력 있는 응답                 | 비유, 스토리, 창의적 표현            | 브레인스토밍, 창작, 재미있는 학습    |
| 효율적       | 짧고 핵심만 전달                       | 빠른 결론, 단계별 정리, 체크리스트       | 기술 문제 해결, 트러블슈팅, 업무 체크 |
| 너디 또는 독특함 | 탐구적이고 깊이 있는 설명                  | 개념 확장, 원리 설명, 트레이드오프 분석    | 심화 학습, 실험, 사고 확장       |
| 냉소적       | 건조하고 위트 섞인 직설                   | 날카로운 관찰, 비꼬는 표현            | 강한 피드백이 필요한 브레인스토밍     |

정답이 있는 설정은 아닙니다. 중요한 것은 내가 ChatGPT 를 어떤 목적으로 주로 사용하는지에 따라 선택하는 것입니다.

업무 문서, 보고서, 기획서, 이메일을 주로 작성한다면 `프로페셔널` 또는 `효율적` 스타일이 잘 맞습니다.

아이디어 정리, 블로그 글, 자기계발, 일상 대화에 많이 사용한다면 `친근함` 이나 `엉뚱함` 도 유용할 수 있습니다.

## 5. 특성 설정

기본 스타일 및 말투 아래에는 ChatGPT 응답의 결을 조정하는 특성 항목이 있습니다.

대표적으로 다음과 같은 특성을 조정할 수 있습니다.

* 따뜻함
* 열정적
* 헤더 및 목록 사용
* 이모지 사용

각 항목은 응답 방식에 직접적인 영향을 줍니다.

### 5.1 따뜻함

따뜻함은 ChatGPT 가 얼마나 친근하고 개인적으로 다가오는지를 조정하는 설정입니다.

| 설정 | 응답 경향                 |
| -- | --------------------- |
| 높음 | 친근하고 공감적인 표현이 많아짐     |
| 낮음 | 업무적이고 사실 중심적인 표현이 많아짐 |

업무용으로 사용할 때는 따뜻함을 낮게 설정하는 것이 더 적합할 수 있습니다. 반대로 자기 성찰이나 일상 대화에 사용할 때는 따뜻함을 높이면 더 자연스럽게 느껴질 수 있습니다.

### 5.2 열정적

열정적 설정은 응답의 에너지와 활기를 조정합니다.

| 설정 | 응답 경향               |
| -- | ------------------- |
| 높음 | 적극적이고 활기 있는 표현이 많아짐 |
| 낮음 | 차분하고 중립적인 표현이 많아짐   |

기업 보고서나 공식 문서 초안을 만들 때는 열정적인 표현이 과할 수 있습니다. 반대로 동기부여, 아이디어 발상, 창작 작업에서는 열정적인 톤이 도움이 될 수 있습니다.

### 5.3 헤더 및 목록 사용

헤더 및 목록 사용은 답변을 구조화해서 보여줄지, 문단 중심으로 보여줄지를 조정합니다.

| 설정 | 응답 경향                  |
| -- | ---------------------- |
| 높음 | 제목, 목록, 표, 단계별 구성이 많아짐 |
| 낮음 | 산문형 문단이 많아짐            |

실무 문서나 학습 자료에는 헤더와 목록이 유용합니다. 반면 에세이나 자연스러운 글을 원할 때는 낮게 설정하는 편이 나을 수 있습니다.

### 5.4 이모지 사용

이모지 사용은 응답에 이모지를 얼마나 포함할지 조정합니다.

| 설정 | 응답 경향         |
| -- | ------------- |
| 높음 | 이모지를 적극적으로 사용 |
| 낮음 | 이모지 사용을 줄임    |

업무용 문서, 공식 메일, 보고서 초안에는 이모지를 낮게 설정하는 것이 일반적으로 적합합니다. 반대로 블로그, SNS, 가벼운 학습 콘텐츠에는 이모지를 높게 설정할 수도 있습니다.

맞춤 설정은 상황별로 다르게 사용할 수 있습니다.

업무 문서 작성, 보고서 작성, 회의록 정리, 고객 커뮤니케이션에 주로 사용한다면 다음 조합이 적합합니다.

이 조합은 불필요한 설명을 줄이고 바로 실행 가능한 답변을 얻는 데 적합합니다.

### 6.3 블로그와 SNS 용 추천 조합

블로그 글, SNS 게시물, 개인 콘텐츠 작성에 활용하고 싶다면 다음 조합을 사용할 수 있습니다.

| 항목          | 추천 설정    |

이 조합은 자연스럽고 대화체에 가까운 답변을 얻는 데 도움이 됩니다.

### 6.4 브레인스토밍용 추천 조합

아이디어 발산, 창작, 새로운 관점이 필요한 경우에는 다음 조합을 고려할 수 있습니다.

| 항목          | 추천 설정     |

이 조합은 예측 가능한 답변보다 다양한 관점과 표현을 얻고 싶을 때 적합합니다.

## 7. 맞춤형 지침

### 7.1 맞춤형 지침이란 무엇인가

앞에서 살펴본 스타일과 특성이 일종의 프리셋이라면, **맞춤형 지침** 은 그 프리셋만으로 표현하기 어려운 세부 지시를 직접 작성하는 공간입니다.

맞춤형 지침에는 ChatGPT 의 행동, 스타일, 말투, 응답 방식에 대한 지침을 입력할 수 있습니다.

예를 들어 다음과 같은 지침을 넣을 수 있습니다.

* 항상 표 형식으로 정리해 주세요.
* 답변은 짧고 핵심 위주로 작성해 주세요.
* 초보자도 이해할 수 있게 설명해 주세요.
* 한국어로 답변해 주세요.
* 업무 문서에 맞게 격식 있는 표현을 사용해 주세요.
* 답변 마지막에 다음에 물어볼 만한 질문을 제안해 주세요.

즉, 매번 반복해서 요청하지 않아도 되는 기본 동작 방식을 미리 지정해 두는 기능입니다.

### 7.2 맞춤형 지침 작성 시 주의할 점

맞춤형 지침은 너무 길고 복잡하게 작성하면 오히려 효과가 떨어질 수 있습니다.

좋은 맞춤형 지침은 다음 기준을 충족합니다.

* 자주 반복하는 요청을 담고 있다.
* 답변 스타일과 형식을 명확히 지정한다.
* 너무 많은 규칙을 한꺼번에 넣지 않는다.
* 서로 충돌하는 지시를 넣지 않는다.
* 실제로 자주 사용하는 업무 상황에 맞춰 작성한다.

예를 들어 다음과 같은 지침은 좋습니다.

```text
답변은 한국어로 작성해 주세요.
업무 문서에 사용할 수 있도록 명확하고 정제된 표현을 사용해 주세요.
복잡한 내용은 표나 단계별 목록으로 정리해 주세요.
필요한 경우 예시를 함께 제시해 주세요.
```

반대로 다음과 같은 지침은 충돌이 생길 수 있습니다.

```text
항상 매우 짧게 답변해 주세요.
동시에 모든 배경 설명과 예시를 자세히 포함해 주세요.
```

짧게 답하라는 지시와 자세히 설명하라는 지시가 충돌하기 때문입니다.

## 8. 추천 맞춤형 지침 예시

강의에서 추천하는 맞춤형 지침은 **후속 질문 3개를 자동으로 제안하게 하는 지침** 입니다.

이 지침을 넣어 두면 ChatGPT 가 답변한 뒤, 이어서 물어보면 좋을 질문을 자동으로 제안합니다. 덕분에 사용자는 꼬리에 꼬리를 무는 방식으로 대화를 이어갈 수 있습니다.

아래 내용을 맞춤형 지침에 넣어 보세요.

```text
응답 후에는 제가 질문하는 형식의 후속 질문 3개를 제공해 주세요.

각 질문은 Q1, Q2, Q3 형식의 굵은 글씨로 표시하고 줄을 바꿔서 표기해 주세요.

이 질문들은 생각을 확장시키고 원래 주제를 더 깊이 탐구할 수 있는 내용이어야 합니다.
```

저장한 뒤 새 채팅에서 아래 질문을 입력해 보세요.

```text
앞으로 5년 동안 직장인의 업무 방식은 어떻게 바뀔까요?
```

맞춤형 지침이 잘 적용되었다면, ChatGPT 는 답변 마지막에 후속 질문 3개를 함께 제안합니다.

예를 들면 다음과 같은 형태입니다.

```text
Q1. 앞으로 5년 동안 가장 크게 사라질 가능성이 있는 직무와 새롭게 생길 직무는 무엇일까요?

Q2. AI 시대에 직장인이 반드시 익혀야 할 핵심 역량 다섯 가지는 무엇일까요?

Q3. 내가 지금 맡고 있는 업무에서 AI 를 가장 먼저 적용해 볼 수 있는 영역은 어디일까요?
```

이제 마음에 드는 질문이 있다면 전체 문장을 다시 입력할 필요 없이, 해당 질문을 복사하거나 번호를 기준으로 이어서 요청할 수 있습니다.

이 방식은 ChatGPT 와의 대화를 더 깊게 확장하는 데 매우 유용합니다.

## 9. 사용자 정보 입력

개인 맞춤 설정에는 사용자가 자신에 대해 알려줄 수 있는 항목이 있습니다.

대표적으로 다음 세 가지가 있습니다.

* 닉네임
* 직업
* 추가 정보

### 9.1 닉네임

닉네임은 ChatGPT 가 사용자를 어떻게 불러야 할지 알려주는 항목입니다.

예를 들어 `최반장` 이라고 입력하면 ChatGPT 는 사용자를 `최반장님` 처럼 부를 수 있습니다.

닉네임은 꼭 실명을 입력할 필요는 없습니다. 본인이 편하게 느끼는 이름이나 업무상 별칭을 사용할 수 있습니다.

### 9.2 직업

직업은 ChatGPT 가 답변을 개인화하는 데 매우 중요한 정보입니다.

가능하다면 단순히 “회사원” 이라고 쓰기보다, 조금 더 구체적으로 작성하는 것이 좋습니다.

예를 들어 다음과 같이 작성할 수 있습니다.

```text
저는 기업 교육 강사이며, Excel, Power BI, 생성형 AI 의 실무 활용을 교육하고 컨설팅합니다.
```

또는 다음처럼 작성할 수도 있습니다.

```text
저는 제조업 기획팀에서 일하고 있으며, 보고서 작성, 데이터 분석, 회의 자료 준비 업무를 자주 합니다.
```

직업 정보를 구체적으로 입력하면 ChatGPT 는 더 적절한 예시와 업무 맥락을 반영할 수 있습니다.

### 9.3 추가 정보

추가 정보에는 사용자의 관심사, 선호도, 자주 사용하는 도구, 답변 형식에 대한 선호 등을 입력할 수 있습니다.

예를 들어 다음과 같은 정보를 넣을 수 있습니다.

```text
저는 업무 생산성 향상에 관심이 많습니다.
Excel, Power BI, Power Query, ChatGPT 를 자주 사용합니다.
답변은 가능하면 실무 예시와 함께 설명해 주세요.
한국어로 답변해 주세요.
```

추가 정보는 ChatGPT 가 답변의 방향을 잡는 데 도움을 줍니다.

다만 다음 정보는 입력하지 않는 것이 좋습니다.

* 비밀번호
* 인증번호
* 주민등록번호
* 계좌번호
* 신용카드 정보
* 회사 기밀
* 고객 개인정보
* 내부 시스템 접속 정보
* 공개되면 안 되는 계약 조건

개인화가 강력해질수록 입력 정보의 관리도 중요해집니다. ChatGPT 가 더 잘 답변하도록 돕는 정보와, 절대 입력하면 안 되는 민감정보를 구분해야 합니다.

## 10. 메모리와 맞춤 설정의 차이

맞춤 설정과 함께 자주 등장하는 기능이 **메모리** 입니다.

맞춤 설정은 사용자가 직접 입력해 두는 기본 설정에 가깝습니다. 반면 메모리는 ChatGPT 가 대화 중 중요하다고 판단한 정보를 기억해 두었다가 이후 대화에서 활용하는 기능입니다.

둘의 차이는 다음과 같습니다.

| 구분    | 맞춤 설정                      | 메모리                          |
| ----- | -------------------------- | ---------------------------- |
| 입력 방식 | 사용자가 설정 화면에 직접 입력          | 대화 중 사용자가 알려주거나 ChatGPT 가 기억 |
| 역할    | 기본 응답 방식과 사용자 정보를 지정       | 대화가 바뀌어도 중요한 정보를 유지          |
| 예시    | “답변은 한국어로 해줘”, “나는 교육 강사야” | “사용자는 Power BI 강의를 자주 진행함”   |
| 관리 위치 | 개인 맞춤 설정 화면                | 메모리 관리 화면                    |
| 목적    | 기본 동작 방식 지정                | 장기적 개인화                      |

예를 들어 맞춤 설정에 “나는 데이터 분석 강사입니다” 라고 입력해 두면, ChatGPT 는 답변할 때 교육과 데이터 분석 맥락을 더 잘 반영할 수 있습니다.

반면 대화 중에 “나는 Power BI 교육을 자주 해” 라고 말했고, ChatGPT 가 이 정보를 메모리로 저장했다면, 이후 다른 대화에서도 그 정보를 참고할 수 있습니다.

메모리는 별도 강의에서 더 자세히 다룹니다. 이번 강의에서는 맞춤 설정과 메모리가 모두 개인화를 위한 기능이라는 점만 기억하면 됩니다.

## 11. 실습하기

이번 강의에서는 설정을 바꿔가며 응답 차이를 직접 확인해 보는 것이 중요합니다.

### 11.1 스타일별 응답 비교

기본 스타일 및 말투를 바꿔가며 아래 프롬프트를 입력해 보세요.

```text
나의 직무와 직업과 관련해서 ChatGPT 를 어떻게 사용할 수 있을까요?
```

비교할 포인트는 다음과 같습니다.

* 답변의 말투가 어떻게 달라지는가?
* 예시가 더 실무적인가, 더 친근한가?
* 표나 목록을 많이 사용하는가?
* 설명이 짧은가, 자세한가?
* 내가 선호하는 업무 방식에 잘 맞는가?

### 11.2 맞춤형 지침 적용 확인

맞춤형 지침에 후속 질문 3개 규칙을 입력한 뒤 아래 질문을 입력해 보세요.

```text
ChatGPT 를 업무 생산성 향상에 활용하려면 무엇부터 시작해야 하나요?
```

응답 마지막에 `Q1`, `Q2`, `Q3` 형식의 후속 질문이 붙는지 확인합니다.

### 11.3 닉네임 적용 확인

닉네임을 설정한 뒤 새 채팅에서 아래처럼 입력해 보세요.

```text
안녕하세요. 반갑습니다.
```

ChatGPT 가 설정한 닉네임으로 불러준다면 닉네임 설정이 정상적으로 반영된 것입니다.

### 11.4 메모리 확인

메모리가 켜져 있다면 아래 프롬프트로 ChatGPT 가 나에 대해 어떤 정보를 기억하고 있는지 확인할 수 있습니다.

```text
나에 대해 무엇을 기억하고 있어?
```

기억된 정보가 잘못되었거나 더 이상 필요 없다면 설정의 메모리 관리 화면에서 삭제할 수 있습니다.

## 12. 실무에서 사용할 때의 주의사항

맞춤 설정은 ChatGPT 의 품질을 높이는 강력한 기능입니다. 하지만 그만큼 주의해야 할 점도 있습니다.

### 12.1 하나의 계정은 한 사람이 사용하는 것이 좋습니다

ChatGPT 는 사용자 정보를 바탕으로 개인화됩니다.

만약 가족, 친구, 동료가 하나의 계정을 함께 사용하면 서로 다른 정보가 섞일 수 있습니다. 그러면 ChatGPT 가 잘못된 맥락으로 답변할 가능성이 높아집니다.

예를 들어 한 사람은 마케팅 업무를 하고, 다른 사람은 개발 업무를 하는데 같은 계정을 사용한다면 ChatGPT 는 어느 맥락에 맞춰 답변해야 할지 혼란스러울 수 있습니다.

따라서 가능하면 하나의 ChatGPT 계정은 한 사람이 사용하는 것이 좋습니다.

### 12.2 민감정보는 입력하지 않습니다

맞춤 설정에는 사용자의 직업, 관심사, 선호도를 입력할 수 있습니다. 하지만 다음과 같은 정보는 절대 입력하지 않습니다.

* 비밀번호
* API 키
* 주민등록번호
* 여권번호
* 계좌번호
* 신용카드 번호
* 고객 개인정보
* 회사 기밀
* 내부 시스템 접속 정보
* 비공개 계약 조건

개인화가 잘될수록 ChatGPT 의 응답 품질은 좋아질 수 있습니다. 하지만 민감정보까지 입력해야 한다는 뜻은 아닙니다.

### 12.3 설정은 목적에 맞게 바꿉니다

맞춤 설정은 한 번 정하면 끝나는 것이 아닙니다.

업무용으로 사용할 때와 개인적인 글쓰기에 사용할 때, 학습용으로 사용할 때, 브레인스토밍에 사용할 때 적합한 설정이 다를 수 있습니다.

따라서 일정 기간 사용해 본 뒤 내 목적에 맞게 조정하는 것이 좋습니다.

## 13. 핵심 정리

* 맞춤 설정의 목적은 ChatGPT 를 나에게 맞게 개인화하는 것입니다.
* 기본 스타일 및 말투는 ChatGPT 가 어떤 분위기와 성격으로 응답할지 정하는 설정입니다.
* 따뜻함, 열정적, 헤더 및 목록 사용, 이모지 사용 같은 특성은 응답의 결을 조정합니다.
* 맞춤형 지침은 매번 반복하지 않아도 되는 기본 응답 규칙을 지정하는 기능입니다.
* 후속 질문 3개를 자동으로 제안하게 하는 지침은 대화를 깊게 확장하는 데 유용합니다.
* 닉네임, 직업, 추가 정보를 입력하면 ChatGPT 의 답변이 더 개인화될 수 있습니다.
* 메모리는 대화 중 알게 된 사용자 정보를 이후 대화에서도 참고할 수 있게 하는 기능입니다.
* 계정은 가능하면 한 사람이 사용하는 것이 좋습니다.
* 비밀번호, 계좌번호, 회사 기밀, 고객 개인정보 같은 민감정보는 맞춤 설정에 입력하지 않습니다.
* 맞춤 설정은 정답이 있는 것이 아니라, 사용 목적과 업무 환경에 맞게 조정하는 기능입니다.
