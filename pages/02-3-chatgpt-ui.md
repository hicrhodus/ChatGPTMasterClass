# 02-3. ChatGPT UI

## 1. 이 강의에서 배울 내용

이번 강의에서는 ChatGPT 의 화면 구성과 기본 사용 흐름을 살펴봅니다.

ChatGPT 를 잘 사용하려면 프롬프트 작성법만 알아서는 부족합니다. 내가 지금 어느 화면에 있는지, 어떤 버튼이 어떤 역할을 하는지, 대화가 어디에 저장되고 어떻게 다시 찾는지 이해해야 합니다.

이 강의를 통해 다음 내용을 익힐 수 있습니다.

* ChatGPT 의 기본 화면 구조를 이해할 수 있습니다.
* 사이드바와 대화 화면의 역할을 구분할 수 있습니다.
* 새 채팅, 채팅 검색, 설정 메뉴의 쓰임새를 이해할 수 있습니다.
* 모델 선택기와 임시 채팅의 의미를 이해할 수 있습니다.
* 프롬프트 입력창과 도구 버튼의 기본 사용 흐름을 익힐 수 있습니다.
* 음성 입력과 음성 모드의 차이를 구분할 수 있습니다.

## 2. 왜 UI 를 먼저 이해해야 하는가

기업 실무자 교육을 진행하다 보면 자주 듣는 후기가 있습니다.

> 숨겨진 기능을 알게 되어서 좋았습니다.

교육 내용이 도움이 되었다는 뜻이라 감사한 말이지만, 한편으로는 흥미로운 표현이기도 합니다. 정말 기능이 숨겨져 있었을까요?

대부분의 경우 프로그램을 만든 회사가 애써 만든 기능을 일부러 숨길 이유는 없습니다. 오히려 사용자가 기능을 잘 발견하고 사용할 수 있도록 화면을 설계합니다.

그런데 우리가 자주 쓰는 기능만 계속 사용하다 보면, 그 외의 기능은 원래 없었던 것처럼 느껴집니다. 그러다 교육이나 설명을 통해 새로운 버튼을 발견하면 “숨겨진 기능”처럼 느끼게 됩니다.

ChatGPT 도 마찬가지입니다.

처음에는 입력창에 질문을 쓰고 답변을 받는 정도로만 사용하기 쉽습니다. 하지만 화면을 자세히 보면 대화 관리, 검색, 설정, 모델 선택, 임시 채팅, 파일 첨부, 도구 실행, 음성 기능 등 다양한 기능이 배치되어 있습니다.

이 강의에서는 ChatGPT 의 UI 를 하나씩 살펴보면서, 어디에서 무엇을 할 수 있는지 정리해 보겠습니다.

## 3. ChatGPT 기본 화면 구조

이번 강의는 ChatGPT 웹 버전을 기준으로 설명합니다.

모바일 앱과 PC 용 앱도 세부 배치는 조금 다를 수 있지만, 기본 개념은 거의 비슷합니다. 따라서 웹 버전의 구조를 이해해 두면 다른 환경에서도 쉽게 적응할 수 있습니다.

ChatGPT 화면은 크게 두 영역으로 나눌 수 있습니다.

| 영역    | 역할                                 |
| ----- | ---------------------------------- |
| 사이드바  | 새 채팅 시작, 대화 목록 확인, 채팅 검색, 설정 메뉴 접근 |
| 대화 화면 | 실제 프롬프트 입력, ChatGPT 답변 확인, 도구 사용   |

왼쪽에는 사이드바가 있고, 오른쪽에는 실제 대화 화면이 있습니다.

대화를 작성하고 답변을 확인하는 공간은 오른쪽 대화 화면입니다. 반면 왼쪽 사이드바에서는 새 대화를 만들거나, 이전 대화를 찾거나, 설정 메뉴에 접근할 수 있습니다.

즉, ChatGPT 화면을 이해할 때는 이렇게 기억하면 됩니다.

> 오른쪽은 작업 공간, 왼쪽은 대화와 메뉴를 관리하는 공간입니다.

## 4. 사이드바

사이드바는 ChatGPT 화면의 왼쪽에 있는 영역입니다.

이곳에서는 대화 목록을 보고, 새 채팅을 시작하고, 이전 대화를 검색하고, 설정 메뉴에 접근할 수 있습니다.

사이드바는 필요에 따라 접거나 펼칠 수 있습니다. 화면을 넓게 사용하고 싶다면 사이드바를 닫고, 이전 대화를 찾거나 메뉴를 사용할 때는 다시 열면 됩니다.

### 4.1 사이드바 닫기와 열기

사이드바 닫기 버튼을 누르면 왼쪽 영역이 최소화됩니다.

이렇게 하면 오른쪽 대화 화면이 더 넓어져서 긴 답변을 보거나 문서를 작성할 때 편리합니다. 다시 사이드바 열기 버튼을 누르면 사이드바가 펼쳐집니다.

정리하면 다음과 같습니다.

* 작업 화면을 넓게 쓰고 싶을 때는 사이드바를 닫습니다.
* 이전 대화를 찾거나 메뉴를 사용할 때는 사이드바를 엽니다.
* 화면이 좁은 노트북이나 태블릿에서는 사이드바 닫기가 특히 유용합니다.

### 4.2 새 채팅

사이드바에서 가장 자주 사용하는 기능 중 하나가 **새 채팅** 입니다.

새 채팅은 말 그대로 새로운 대화를 시작하는 기능입니다.

ChatGPT 는 하나의 채팅 안에서 앞서 나눈 대화의 맥락을 이어받습니다. 따라서 같은 주제나 같은 목표의 대화라면 하나의 채팅 안에서 계속 이어가는 것이 좋습니다.

반대로 주제나 목적이 완전히 달라졌다면 새 채팅을 시작하는 것이 좋습니다.

예를 들어 다음과 같은 경우에는 새 채팅을 권장합니다.

* Excel 함수 질문을 하다가 여행 계획을 세우고 싶을 때
* 보고서 초안을 작성하다가 전혀 다른 이메일 문구를 만들고 싶을 때
* 고객사 제안서 작업과 개인 학습 질문을 분리하고 싶을 때
* 이전 대화의 맥락이 새 작업에 영향을 주지 않게 하고 싶을 때

새 채팅은 새로운 대화방을 만드는 기능이라고 이해하면 됩니다.

단축키는 다음과 같습니다.

```text
Ctrl + Shift + O
```

ChatGPT 를 자주 사용한다면 새 채팅 단축키는 기억해 두는 것이 좋습니다.

### 4.3 채팅 검색

ChatGPT 를 계속 사용하다 보면 대화가 점점 많아집니다.

보고서 작성, 이메일 초안, 아이디어 정리, 데이터 분석 질문, 강의 기획 등 다양한 대화가 쌓이면 필요한 대화를 다시 찾기 어려워집니다.

이때 사용하는 기능이 **채팅 검색** 입니다.

채팅 검색을 사용하면 이전 대화의 제목이나 내용에서 키워드를 기준으로 검색할 수 있습니다.

예를 들어 `학습`, `보고서`, `Power BI`, `프롬프트`, `고객사 제안` 같은 단어를 검색하면 관련 대화를 빠르게 찾을 수 있습니다.

채팅 검색 단축키는 다음과 같습니다.

```text
Ctrl + K
```

대화가 많아질수록 검색 기능의 중요성이 커집니다. ChatGPT 를 업무 도구로 사용한다면, 새 채팅 단축키와 함께 채팅 검색 단축키도 익혀 두는 것이 좋습니다.

### 4.4 설정 메뉴

화면 왼쪽 하단에는 현재 로그인한 계정 아이콘이 있습니다.

이 계정 아이콘을 클릭하면 여러 가지 메뉴가 나타납니다. 여기에서 설정 메뉴에 접근할 수 있습니다.

설정 메뉴에서는 ChatGPT 사용 환경을 조정할 수 있습니다.

대표적으로 다음과 같은 항목을 확인할 수 있습니다.

* 일반 설정
* 알림
* 개인 맞춤 설정
* 데이터 관련 설정
* 화면 표시 방식
* 계정 및 플랜 관련 메뉴

이번 강의에서는 설정 메뉴를 하나씩 자세히 다루지는 않습니다. 다만 설정 메뉴는 ChatGPT 를 나에게 맞게 조정하는 중요한 공간이라는 점만 먼저 기억하면 됩니다.

특히 개인 맞춤 설정은 이후 강의에서 별도로 다룰 예정입니다. ChatGPT 가 나의 업무 방식이나 선호하는 답변 스타일을 더 잘 이해하도록 설정할 수 있기 때문에, 실무 활용에서 매우 중요한 기능입니다.

## 5. 모델 선택기

대화 화면 상단에는 ChatGPT 또는 모델 이름이 표시되는 영역이 있습니다. 이 영역을 클릭하면 사용할 수 있는 모델이나 대화 모드를 선택할 수 있습니다.

이 부분을 **모델 선택기** 라고 이해하면 됩니다.

ChatGPT 는 다양한 AI 모델을 바탕으로 작동합니다. 모델은 쉽게 말해 ChatGPT 의 답변 능력을 결정하는 핵심 엔진에 가깝습니다.

비유하자면 다음과 같습니다.

| 비유      | 설명                         |
| ------- | -------------------------- |
| ChatGPT | 자동차                        |
| AI 모델   | 엔진                         |
| 모델 선택기  | 어떤 엔진이나 주행 모드로 달릴지 선택하는 영역 |

과거에는 사용자가 직접 특정 모델명을 선택해서 사용하는 경우가 많았습니다. 현재는 사용자의 플랜과 작업 성격에 따라 ChatGPT 가 적절한 모델이나 모드를 연결해 주는 방식도 함께 사용됩니다.

화면이나 모델 이름은 시간이 지나면서 바뀔 수 있습니다. 따라서 중요한 것은 특정 모델명을 외우는 것이 아닙니다.

중요한 것은 **이 위치에서 대화 방식이나 모델을 선택할 수 있다** 는 구조를 이해하는 것입니다.

### 5.1 빠른 답변이 필요할 때

일반적인 질문이나 가벼운 대화, 빠른 요약처럼 즉시 답변이 필요한 작업에는 빠른 응답 중심의 모드가 적합합니다.

예를 들어 다음과 같은 작업입니다.

* 짧은 문장 다듬기
* 간단한 개념 설명
* 이메일 제목 추천
* 아이디어 몇 가지 뽑기
* 짧은 요약 요청

### 5.2 깊이 있는 사고가 필요할 때

복잡한 분석, 여러 조건 비교, 전략적 판단, 논리적 추론이 필요한 작업에는 더 깊게 생각하는 모드가 적합합니다.

예를 들어 다음과 같은 작업입니다.

* 보고서 구조 설계
* 여러 대안 비교
* 데이터 분석 방향 검토
* 복잡한 의사결정 지원
* 교육 커리큘럼 설계
* 프로젝트 리스크 분석

정리하면 이렇게 이해하면 됩니다.

> 빠르고 간단한 질문은 빠른 응답 중심으로, 깊이 있는 사고가 필요한 질문은 추론 중심으로 접근합니다.

## 6. 그룹 채팅

화면 오른쪽 상단에는 그룹 채팅을 시작할 수 있는 버튼이 표시될 수 있습니다.

그룹 채팅은 여러 사람이 하나의 ChatGPT 대화 공간에 함께 참여할 수 있는 기능입니다. 링크를 만들어 다른 사람에게 전달하면 여러 명이 하나의 대화방에서 함께 대화할 수 있습니다.

일반적인 단체 채팅방과 다른 점은, 그 공간 안에 ChatGPT 도 함께 참여한다는 것입니다.

즉, 그룹 채팅은 사람들끼리만 대화하는 공간이 아니라 **ChatGPT 가 함께 참여하는 협업 공간** 입니다.

예를 들어 다음과 같은 상황에서 활용할 수 있습니다.

* 회의 준비
* 아이디어 회의
* 문서 검토
* 팀 단위 브레인스토밍
* 교육 기획 논의
* 프로젝트 요구사항 정리

다만 그룹 채팅은 별도의 강의에서 더 자세히 다룰 예정입니다. 지금은 화면 안에 이런 협업 기능이 있다는 정도로 이해하면 됩니다.

## 7. 임시 채팅

임시 채팅은 일반 대화 기록에 남기지 않고 일회성으로 대화를 나누고 싶을 때 사용하는 기능입니다.

임시 채팅을 켜면 일반 대화와 다르게 처리됩니다. 이전 대화나 메모리를 참조하지 않는 상태에서 대화를 시작할 수 있고, 대화 기록에 남지 않는 방식으로 사용할 수 있습니다.

임시 채팅은 다음과 같은 상황에 적합합니다.

* 잠깐 확인하고 끝낼 질문
* 대화 기록에 남길 필요가 없는 요청
* 일회성 아이디어 확인
* 기존 메모리나 이전 대화의 영향을 받지 않고 답변을 받고 싶을 때
* 테스트성 질문을 해보고 싶을 때

하지만 주의해야 할 점도 있습니다.

임시 채팅이라고 해서 민감한 개인정보나 회사 기밀을 무분별하게 입력해도 된다는 뜻은 아닙니다. 업무 환경에서는 항상 조직의 보안 정책과 데이터 관리 기준을 함께 고려해야 합니다.

정리하면 다음과 같습니다.

> 임시 채팅은 기록을 남기지 않고 가볍게 대화하고 싶을 때 사용하는 기능입니다. 하지만 보안 기준을 무시해도 되는 기능은 아닙니다.

## 8. 프롬프트 입력창

대화 화면의 가운데 또는 아래쪽에는 입력창이 있습니다. 이곳이 ChatGPT 에게 질문과 요청을 입력하는 공간입니다.

이 입력창을 **프롬프트 입력창** 이라고 부를 수 있습니다.

우리가 ChatGPT 에게 보내는 질문, 요청, 지시문을 프롬프트라고 합니다. 따라서 프롬프트 입력창은 ChatGPT 사용의 중심 공간입니다.

프롬프트 입력창에서는 다음과 같은 작업을 할 수 있습니다.

* 질문 입력
* 요청 작성
* 긴 설명 입력
* 파일 첨부
* 도구 실행
* 음성 입력
* 음성 모드 시작
* 결과물 생성을 위한 지시 작성

즉, 프롬프트 입력창은 단순히 글자를 입력하는 칸이 아닙니다. ChatGPT 에게 작업을 지시하는 중심 공간입니다.

## 9. 도구 버튼

프롬프트 입력창 주변에는 도구를 선택할 수 있는 버튼이 있습니다. 일반적으로 플러스 버튼이나 도구 메뉴 형태로 표시될 수 있습니다.

이 버튼을 누르면 ChatGPT 를 단순한 대화 도구가 아니라 다양한 작업 도구와 함께 사용할 수 있습니다.

예를 들어 다음과 같은 기능을 사용할 수 있습니다.

* 사진 또는 파일 추가
* 최근 사용한 파일 확인
* 이미지 만들기
* 웹 검색
* 심층 리서치
* 에이전트 모드
* 캔버스 또는 문서 작업 관련 기능
* 퀴즈나 학습 관련 기능

여기서 중요한 점은 **도구를 먼저 선택하면 ChatGPT 가 내가 원하는 작업의 종류를 더 명확하게 이해한다** 는 것입니다.

예를 들어 이미지 만들기 도구를 선택한 뒤 다음처럼 입력한다고 해보겠습니다.

```text
포효하고 있는 사자
```

그냥 보면 완전한 문장이 아닙니다. 하지만 이미지 만들기 도구를 선택한 상태라면 ChatGPT 는 이 요청을 이미지 생성 요청으로 이해할 수 있습니다.

이처럼 도구 버튼은 ChatGPT 에게 작업의 방향을 알려주는 역할을 합니다.

각 도구는 내용이 방대하므로 이번 강의에서 모두 깊게 다루지는 않습니다. 이후 강의에서 파일 추가, 웹 검색, 심층 리서치, 이미지 생성, 에이전트 모드 등을 각각 자세히 살펴보겠습니다.

## 10. 음성 입력

프롬프트 입력창 오른쪽에는 마이크 모양 아이콘이 표시될 수 있습니다.

이 기능은 **음성 입력** 입니다.

음성 입력은 키보드로 직접 입력하는 대신 말로 프롬프트를 입력하는 기능입니다. 마이크 버튼을 누르고 말하면, ChatGPT 가 내가 말한 내용을 텍스트로 받아 적어 프롬프트 입력창에 넣어 줍니다.

즉, 음성 입력은 받아쓰기 기능에 가깝습니다.

이 기능은 다음과 같은 상황에서 유용합니다.

* 긴 문장을 타이핑하기 어려울 때
* 이동 중에 아이디어를 빠르게 기록하고 싶을 때
* 회의 내용을 바탕으로 간단히 정리하고 싶을 때
* 말로 설명하는 것이 글로 쓰는 것보다 편할 때
* 모바일에서 긴 프롬프트를 작성해야 할 때

다만 주변 소음이 많거나 마이크 품질이 좋지 않으면 인식 결과가 부정확할 수 있습니다. 중요한 업무 문서에 활용할 때는 음성 입력 후 결과를 반드시 확인하고 수정해야 합니다.

음성 입력은 이후 강의에서 더 자세히 다룹니다.

## 11. 음성 모드

마이크 아이콘과 별도로 음성 모드를 시작하는 버튼이 표시될 수 있습니다.

음성 입력과 음성 모드는 서로 다릅니다.

| 구분     | 음성 입력               | 음성 모드                   |
| ------ | ------------------- | ----------------------- |
| 목적     | 말한 내용을 텍스트로 입력      | ChatGPT 와 음성으로 대화       |
| 방식     | 받아쓰기                | 실시간 음성 대화               |
| 결과     | 입력창에 텍스트가 들어감       | 말로 질문하고 말로 답변을 받음       |
| 적합한 상황 | 긴 프롬프트 작성, 업무 요청 정리 | 이동 중 대화, 아이디어 발산, 회화 연습 |

음성 모드를 켜면 내가 말을 하고, ChatGPT 가 음성으로 답합니다. 필요하면 중간에 말을 끊고 다시 질문할 수도 있습니다.

이런 방식은 텍스트 채팅보다 훨씬 실제 대화에 가까운 경험을 제공합니다.

음성 모드는 다음 상황에서 유용합니다.

* 이동 중 이어폰을 끼고 사용할 때
* 손으로 타이핑하기 어려운 상황
* 아이디어를 말로 정리하고 싶을 때
* 외국어 회화 연습을 하고 싶을 때
* 빠르게 질의응답을 주고받고 싶을 때

다만 운전 중이거나 주변 주의가 필요한 상황에서는 항상 안전을 먼저 고려해야 합니다.

## 12. 전송 버튼과 Enter 키

프롬프트 입력창에 아무것도 입력되어 있지 않은 상태에서는 오른쪽 끝에 음성 모드 버튼이 보일 수 있습니다.

하지만 입력창에 텍스트를 입력하는 순간, 이 버튼이 전송 버튼으로 바뀝니다.

예를 들어 입력창에 다음처럼 입력해 보겠습니다.

```text
안녕하세요.
```

이제 오른쪽 버튼은 전송 버튼으로 바뀝니다. 이 버튼을 누르면 작성한 프롬프트가 ChatGPT 에 전달됩니다.

또는 키보드의 `Enter` 키를 눌러도 프롬프트를 전송할 수 있습니다.

정리하면 프롬프트를 보내는 방법은 두 가지입니다.

* 전송 버튼을 누른다.
* `Enter` 키를 누른다.

긴 문장을 작성하거나 줄바꿈이 필요한 경우에는 사용 환경에 따라 `Shift + Enter` 로 줄바꿈을 사용할 수 있습니다.

중요한 것은 입력창의 상태에 따라 오른쪽 버튼의 역할이 바뀐다는 점입니다.

* 아무것도 입력하지 않았을 때: 음성 모드 버튼
* 텍스트를 입력했을 때: 전송 버튼

이 구조를 이해하면 프롬프트 입력창을 더 자연스럽게 사용할 수 있습니다.

## 13. 핵심 정리

* ChatGPT 화면은 크게 왼쪽 사이드바와 오른쪽 대화 화면으로 나눌 수 있습니다.
* 사이드바는 새 채팅, 채팅 검색, 설정 메뉴 등 대화와 메뉴를 관리하는 공간입니다.
* 오른쪽 대화 화면은 실제로 프롬프트를 입력하고 ChatGPT 와 대화하는 작업 공간입니다.
* 주제나 목적이 바뀌면 새 채팅을 시작해 대화 맥락을 분리하는 것이 좋습니다.
* 이전 대화를 찾을 때는 채팅 검색을 사용합니다.
* 설정 메뉴에서는 ChatGPT 를 나에게 맞게 조정할 수 있습니다.
* 모델 선택기는 대화 방식이나 모델을 선택하는 위치입니다.
* 임시 채팅은 기록을 남기지 않고 가볍게 대화하고 싶을 때 사용합니다.
* 프롬프트 입력창은 ChatGPT 에게 작업을 지시하는 중심 공간입니다.
* 도구 버튼을 사용하면 파일, 이미지, 웹 검색, 리서치 등 다양한 작업을 실행할 수 있습니다.
* 음성 입력은 말을 텍스트로 바꾸는 기능이고, 음성 모드는 ChatGPT 와 음성으로 대화하는 기능입니다.
* 프롬프트를 입력하면 전송 버튼이 나타나며, `Enter` 키로도 전송할 수 있습니다.
